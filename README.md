# HR Attendance & Meeting Dashboard (HROS)

A modern Human Resource Operating System (HROS) featuring a comprehensive attendance tracker, task manager, meeting scheduler, and dashboard.

## 🛠️ Technology Stack & Architecture

This repository is organized as a **PNPM Monorepo** containing the following workspaces:
- **`artifacts/api-server`**: Express.js backend with PostgreSQL, using Drizzle ORM and Pino logging.
- **`artifacts/hr-dashboard`**: React frontend powered by Vite, Tailwind CSS, shadcn/ui components, and React Query.
- **`lib/db`**: Database schemas, connections, and migrations managed using Drizzle.
- **`lib/api-spec`**: OpenAPI specifications and code generator configurations.
- **`lib/api-zod`**: Autogenerated Zod validation schemas.
- **`lib/api-client-react`**: Autogenerated React Query client hooks.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- PNPM (v9+)
- PostgreSQL database running locally or remotely

### 1. Install Dependencies
Run the following command at the root of the project to install all workspace dependencies:
```bash
pnpm install
```

### 2. Configure Environment Variables
You will need a running PostgreSQL database. Set the following environment variable or create a `.env` file in the root / packages:
```bash
DATABASE_URL=postgresql://postgres:postgres@localhost:5432/postgres
```

### 3. Push Database Schema
Push the Drizzle schema to your database instance:
```bash
pnpm --filter @workspace/db run push
```

### 4. Running the Development Servers

#### Start Backend API Server
Runs on port `8080`:
```bash
$env:PORT="8080"
$env:DATABASE_URL="postgresql://postgres:postgres@localhost:5432/postgres"
pnpm --filter @workspace/api-server run dev
```

#### Start Frontend Dashboard
Runs on port `5173` (proxies `/api` requests to `localhost:8080`):
```bash
$env:PORT="5173"
$env:BASE_PATH="/"
pnpm --filter @workspace/hr-dashboard run dev
```
